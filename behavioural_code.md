# CUR8 × GoodFellas — Behavioural Code & Staff Training Programme

## Overview

Kissa-style venue: cafe by day, bar/performance space by night. Rules flex by time slot and event type. Two audiences: **customers** (what's expected) and **staff** (how to enforce).

## Module 1 — Orientation & Venue Identity

**Goal:** Staff understand *why* the rules exist, not just *what* they are.

### Content
- Kissa concept: cafe warmth by day, performance soul by night
- CUR8 × GoodFellas brand voice: welcoming but clear boundaries
- Customer persona: who's coming, what they expect at each slot
- Staff role: **host + guardian** — warmth first, firmness when needed

### Assessment
- Staff can articulate 3 core venue values from memory
- Staff can name the 4 time slots and primary mode of each

## Module 2 — Time-Slot Rules & Transition Protocol

**Goal:** Staff know exactly which rules apply and when to communicate transitions.

### Content

**Transition Checklist — copy for every shift:**

```
SHIFT START
□ Confirm current slot (A/B/C/D)
□ Confirm tonight's event type (none / acoustic / DJ / private)
□ Post signage updated for slot?
□ ID check kit ready? (for slot D)
□ Drink menu updated for slot?

TRANSITION A→B (11:00)
□ Alcohol board activated
□ Morning laptop zone signs down

TRANSITION B→C (17:00)
□ Kids policy signage down / updated
□ Stage area checked / prepped if evening artist

TRANSITION C→D (21:00)
□ ID check at door ON
□ Performance zone stanchions out
□ Last call time communicated to bar team
□ Noise complaint protocol briefed

TRANSITION D→CLOSE
□ Last call posted + verbal
□ Post-show wind-down begins
□ Re-entry protocol: NO re-entry after door close
```

### Assessment
- Staff can run a transition from memory with checklist
- Staff can answer: "What changes at 21:00 tonight?"

## Module 3 — Customer Code Enforcement — Practical Scenarios

**Goal:** Staff handle common situations without escalating unnecessarily.

### Scenario Cards

**S3-1: Customer on laptop with no headphones, volume too loud**
```
Step 1: Approach casually — "Hey, just a heads-up — headphones needed past this volume, for the space"
Step 2: If no compliance → offer a headphone if spares available
Step 3: If still non-compliant → "I need to ask you to lower it or use headphones — fair?"
Step 4: If still resistance → "I appreciate you're a customer, but this is the house rule. Happy to 
help find a spot where it's less disruptive." → offer alternative seating
Step 5: Escalate to manager if still no resolution
```

**S3-2: Customer bringing outside alcohol in**
```
Step 1: Intercept at door — "Hey! Just a heads-up — outside alcohol isn't permitted here. 
We'll happy to pour you something from our menu"
Step 2: If bag/cooler → don't touch, point to bag storage policy
Step 3: If becomes defensive → "I totally get it, but it's our licence condition. 
Happy to keep it behind the bar till you leave, or you can enjoy it outside on the terrace"
Step 4: Escalate only if aggression
```

**S3-3: Customer filming during live set**
```
Step 1: Approach between songs — "No flash photography is fine, but just a heads-up — 
professional filming isn't permitted without prior booking. Happy to take a pic for you at the break!"
Step 2: Offer to take photo on their phone
Step 3: If professional equipment (DSLR/camcorder) → "This is a content restriction from 
the artist agreement. If you'd like to share, I can ask the promoter"
Step 4: No confrontation — document incident for manager
```

**S3-4: Customer staying past 2-hour morning limit**
```
Step 1: At 1h45m → pass by: "All good here? Just a heads-up — we're peak morning and 
have a few waiting. Happy to let you stay a bit longer if things quiet down"
Step 2: At 2h → approach: "Hey — just so you know, we've hit our 2-hour window. 
No pressure, but we're at capacity. I can put your name down for when a spot opens?"
Step 3: Offer alternatives: terrace, bar seating, takeaway
Step 4: Never forcibly remove — escalate to manager
```

## Module 4 — Performance Night Protocols

**Goal:** Staff execute door, floor, and bar protocols during live shows.

### Door Protocol

```
PERFORMANCE NIGHT — DOOR
□ ID check: every patron, every time — no exceptions
□ Stamp/wristband: 18+ verification
□ Search policy: bags visible, no glass, no external alcohol
□ Capacity count — maintain room count
□ Re-entry: NO re-entry after door close
□ Artist brief: confirm artist has no restricting items (confetti, open flame, etc.)
```

### Bar Protocol

```
PERFORMANCE NIGHT — BAR
□ Order queue: single queue, no approaching bar during live sets
□ Staff call "Next please" — no ad-hoc orders during sets
□ Drink tickets: optional for busy nights
□ Last call: 30 min before close — visually communicated + verbal
□ Post-last-call: no new orders after called
□ Glass policy: plastic cups for floor, glass behind bar only
```

### Floor Protocol

```
PERFORMANCE NIGHT — FLOOR
□ Zone: front-of-stage area no standing during acts
□ Safety: clear exit paths, no bags blocking egress
□ Monitoring: 1 staff eyes on floor at all times during live sets
□ Intervention: any fall, spill, or aggression → signal bar immediately
□ Photos: allow phone, discourage flash, no obstruction of sight lines
```

## Module 5 — Conflict De-escalation

**Goal:** Staff de-escalate without aggression from either side.

### De-escalation Hierarchy

```
LEVEL 1 — Ambient approach
→ Make eye contact, smile, open body language before anything goes wrong
→ Intercept early — catch it before the customer even realises they're border-line

LEVEL 2 — Verbal redirect
→ "I know this can feel [uncomfortable/restrictive], but here's why we do it..."
→ Offer an alternative: seat change, time extension, alternative option
→ Use customer's name if known

LEVEL 3 — Firm boundary
→ "I need you to stop [specific behaviour] — this is the house rule"
→ State consequence once: "If this continues, I'll need to ask you to leave"
→ Do NOT argue — state, pause, wait

LEVEL 4 — Exit request
→ "I'm going to ask you to leave now. I'm not going to debate it further."
→ No threats, no raised voice
→ Manager on standby if needed

LEVEL 5 — Physical threat / aggression
→ Press SOS / alert system
→ Do NOT physically intervene alone
→ Manager + authorities if required
```

### Key phrases to practise:

| Situation | Say this |
|---|---|
| Over capacity | "We're at capacity right now — I'm happy to put you on the list and text you when a spot opens" |
| Volume complaint | "I hear you — volume is set by the artist for this set. Happy to grab you earplugs if it's too much?" |
| Rules complaint | "I hear you — the rules aren't personal, they're here so everyone enjoys the night. I can explain the thinking" |
| Refusing to leave | "I'm not going to argue about this. You need to leave — I'll help you get a taxi/another venue" |
| Drugs suspicion | "I'm not accusing you, but I need you to know — zero tolerance here. I'd like you to show me what's in your hand" |

## Module 6 — Event-Type Delivery

### Acoustic Night Delivery Checklist

```
□ Tables arranged for listening (no centre tables blocking sight lines)
□ Lighting: low, warm — no overhead fluorescents
□ Sound: check monitor levels, ensure performers can hear themselves
□ Set list posted or communicated to floor staff
□ No food service during sets (kitchen quiet protocol)
□ Tip bucket: passed at set break only
□ Phone policy: staff reminder to announce "phones on silent"
```

### DJ Night Delivery Checklist

```
□ Dance floor zone marked (stanchions or floor tape)
□ Plastic cups: pre-staged near dance floor
□ Glass policy enforced — no glass past bar counter edge
□ Capacity: tighter monitoring, 1 staff on door at all times
□ Bouncer/Security: confirmed, on-site 30 min before doors
□ Water station: free water near dance floor
□ Last entry time communicated + enforced
```

### Private Event Delivery Checklist

```
□ Pre-event briefing with host/organiser
□ Custom house rules agreed and printed for door staff
□ Capacity agreed — do not exceed
□ Payment / tab arrangement confirmed
□ External promoter insurance/liability confirmed
□ Staff assigned: 1 manager, floor coverage ratio
□ Cleanup + security deposit agreed post-event
```

## Module 7 — Safety & Emergency

### Emergency Contacts — Display in Back Office

| Situation | Contact |
|---|---|
| Medical emergency | 115 |
| Fire | 114 |
| Police (non-emergency) | Local precinct |
| Venue manager (on-call) | [Manager number] |
| Security company | [Security number] |
| Alcohol licensing body | [Number] |

### Emergency Protocol

```
1. ASSESS — Is anyone in immediate danger?
2. ALERT — Signal bar/staff without causing panic
3. CALL  — Emergency services if needed
4. CONTAIN — Secure the area, don't move injured persons
5. GUIDE — Direct emergency services on arrival
6. DOCUMENT — Incident report within 24 hours
```

### Incident Report Fields

```
Date / Time
Staff on duty
Customer name + ID (if available)
What happened
What action taken
What the outcome was
Follow-up required
```
